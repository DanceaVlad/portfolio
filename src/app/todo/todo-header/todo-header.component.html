This is a showcase of an easy to use Reminder-like app. Users can create reminders, check the ones they have completed or fulfilled.

User can use this app in two ways:
Anonymous: This way the reminders will be reset upon leaving or resetting the page.
Sign in: This way the reminders are stored in a database so they can be accessed anytime, anywhere.
